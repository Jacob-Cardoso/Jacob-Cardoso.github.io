Retro Readers

Python | Pygame | Wordnik API

About
Retro Readers is a game I programmed during the hackathon, Hack The 6ix. The game consists of two overall game modes that aim towards an age group of young children with dyslexia looking to become better readers. The font that's used for the majority of the text is called OpenDyslexic. It's been proven that the font aids in readability for dyslexic readers.

Retro Readers was made in under 34 hours during the competition. It's programmed exclusively in Python with the use of the Pygame module to create the GUI. Another module named random_word was implemented due to its use of the extremely versatile Wordnik API.

Legend Of Zelda: Education Edition

Python | Pygame

About
This program is based around the original Legend Of Zelda video game developed by Nintendo but with a unique twist.

Its story is based around a school setting where the player must solve math problems of varying difficulty to complete the game. Depending on the players' grade level, they can choose a difficulty that changes their questions.

The program's functionality is based on a popular Pygame grid system, which I heavily modified for this project while implementing my own ideas, which solved many of the system's inherent problems its users have been struggling with.

RFID Door Lock

Arduino | Blynk | RFID

About
This project is a smart, deadbolt door lock that utilizes RFID and wireless technology. It was made using an Arduino Uno and various hardware pieces, including an LCD display and servo motor. This was my final project for the eleventh-grade Computer Engineering course I took, where I earned a mark of 100% in the course and on this project.

The lock was made to be reprogrammable for any RFID chip through its UID tag, and it's even able to be used Wirelessely via the Blynk application on mobile devices. The circuit diagrams shown were created in Fritzing.

Rock Paper Scissors Lizard Spock

Java

About
Rock Paper Scissors Lizard Spock is an interactive command-line based game developed in Java. This was my first Java project and it follows a unique structure while keeping track of the player and computers stats from round to round. There are 3 game modes that can be played, best of one, best of three or best of 5.

The program uses a 2D array to store and index through the different text responses a player could receive. The game logic is based on a number system which assigns each type of move a number and determines the winner of each round through a mathematic formula.

Compound Interest Calculator Table

Java

About
This Java program was created as a fast, easy-to-use, and versatile command-line based compound interest calculator. The program asks for a principal investment, interest rate and the investment term the program should use to calculate the result. The output is a neatly formatted table produced by the printf function. The table includes the value of the investment after its been compounded at different intervals.

The table that's outputted eliminates the frequency that other interest calculators have, where users have to check their investments worth multiple times with different compounding periods each time.
